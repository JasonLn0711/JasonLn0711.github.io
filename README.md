# Time Zone Checker

Welcome to the **Time Zone Checker** project! This web application is designed to help users manage events across different time zones with ease and mindfulness. Hosted on [GitHub Pages](https://jasonln0711.github.io), this tool provides intuitive features for scheduling, notifications, and time zone conversions.

## Features

- **Event Management**: Add event names, select dates and times, and specify your current and destination time zones.
- **Destination Time Display**: View the exact time in the destination time zone in the `YYYY/MM/DD HH:mm` format.
- **Time Zone Switching**: Swap the current and destination time zones seamlessly.
- **Notifications**: Set reminders for events:
  - Options include "5 minutes before," "exactly the time," and "5 minutes after."
  - Notifications are listed in chronological order.
- **Countdown Timer**: Live countdown to upcoming notifications.
- **Google Calendar Integration**: Add events directly to your Google Calendar.
- **Wabi-Sabi Aesthetic**: Simple and elegant design inspired by the beauty of imperfection.

## Usage

Visit the application at [https://jasonln0711.github.io](https://jasonln0711.github.io). To use:

1. Enter an event name, select the current and destination time zones, and specify a date and time.
2. Click "Show Destination Time" to see the converted time.
3. Use the "Switch Time Zones" button to swap between the two time zones.
4. Set notifications and track countdowns for event reminders.
5. Add events directly to your Google Calendar.
